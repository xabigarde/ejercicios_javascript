Ejercicio 4

Enunciado: Dado una array de valores de cualquier tipo, devuelve un nuevo array con los elementos del array original en orden inverso, sin utilizar sort() ni reverse(). Ejemplo: 

    [0,1,2,3,4,5] => [5,4,3,2,1,0]
    ["Grut","soy","yo"] => ["yo","soy","Grut"]
